# Multi-Layer Equity Selection Framework
A structured framework for screening, ranking and analyzing equities using financial metrics and AI-assisted research.

## Problem

The stock market contains thousands of listed companies.

The challenge is not analyzing a stock.

The challenge is identifying which stocks deserve analysis.

This project creates a structured process for reducing 5000+ stocks into a focused watchlist.
